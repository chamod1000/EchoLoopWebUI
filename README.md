EcoLoop is a smart waste management web application that connects households with local waste collectors.
It provides real-time truck tracking, a recycling rewards system, and an impact dashboard to promote environmental sustainability.

Local Setup Instructions

Prerequisites
Git
Node.js version 18 or higher
Code editor such as VS Code

Installation Steps

Clone the repository
git clone https://github.com/chamod1000/Advanced-Git-Dev-ops-EchoLoop-.git

Navigate to the project directory
cd Advanced-Git-Dev-ops-EchoLoop-

Install dependencies
npm install

Run the application
Open index.html directly in a browser
Or use Live Server in VS Code

CI/CD Pipeline

Continuous Integration

Continuous Deployment
The application is automatically deployed to Vercel when changes are merged into the main branch.
